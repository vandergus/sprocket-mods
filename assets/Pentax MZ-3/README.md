The gerber files located here are for custom sprocket counter PCBs that create a half-frame film advance in the Pentax MZ-3 and similar cameras. They can be used to order custom PCBs from online fab services. When ordering, use the default options except for board thickness and surface finish. Instead, choose...

* Board Thickness: 0.6 mm
* Surface Finish: ENIG

### Compatible Models

* Pentax MZ-3
* Pentax MZ-5n | ZX-5n
* Pentax MZ-5 | ZX-5
* Pentax MZ-6
* Pentax MZ-L | ZX-L
* Pentax MZ-7 | ZX-7
* Pentax MZ-10 | ZX-10
* Pentax MZ-30 | ZX-30
* Pentax MZ-50 | ZX-50
* Pentax MZ-M | ZX-M
