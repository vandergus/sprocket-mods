There are gerber files available for both half-frame and square-frame film advance. These can be used to order custom PCBs from online fab services. When ordering, stick with default options except for board thickness and surface finish. Instead, choose...

* Board Thickness: 0.6 mm
* Surface Finish: ENIG

### Compatible Models

* Nikon F60
* Nikon N60
* Nikon F50
* Nikon N50
